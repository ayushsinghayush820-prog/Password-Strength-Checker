# Password-Strength-Checker

A simple phython script that checks the strength of a password based on mutliple security criteria.

## What it checks 
- Minimum length
- Uppercase letters
- Lowecase letters
- Digits
- Special Characters

  ##How to run
  1. Install Phython
  2. Run; python password checker.py
  3. Enter your password after prompt run

 ## What I learn 
 building this helped me understand string methods in Phython and basic password security principal
